### Actionable Issues

**1. PORT-05 — Homepage portfolio href fallback (non-critical)**
- `src/pages/index.astro` line 38: `href=/portfolio/${p.data.caseStudySlug || p.id}`
- The `p.id` fallback produces URLs that don't match `portfolioCaseStudies` collection IDs
- Not triggered by current data (all 3 projects have `caseStudySlug` set)
- Fix: only render link when `caseStudySlug` is non-null, matching `PortfolioCard.astro` behavior

**2. SEO-06 — Dead `jsonLd` prop in BaseLayout (cosmetic)**
- `BaseLayout.astro` Props interface declares `jsonLd?: object` but it's never destructured or rendered
- JSON-LD is correctly delivered via dedicated `JsonLdOrganization` and `JsonLdBlogPost` components
- Fix: remove unused prop from interface

**3. LEAD-02 — Requirement text mismatch (documentation)**
- REQUIREMENTS.md says "Nodemailer/SMTP" but implementation uses Resend
- This was a documented intentional substitution (functionally equivalent)
- Fix: update requirement text to reflect actual implementation
