  <action>
1. **src/db/schema.ts** -- Add 8 new columns to the `submissions` table definition, AFTER the existing 8 columns. Keep all existing columns exactly as-is. New columns:
   - `serviceType: text('service_type')` -- nullable
   - `budget: text('budget')` -- nullable
   - `timeline: text('timeline')` -- nullable
   - `leadScore: integer('lead_score')` -- nullable
   - `leadStatus: text('lead_status').notNull().default('new')` -- NOT NULL with SQL-level default (use `.default('new')`, NOT `.$defaultFn()` -- the SQL DEFAULT is required so existing rows get 'new' during ALTER TABLE)
   - `notes: text('notes')` -- nullable
   - `hubspotId: text('hubspot_id')` -- nullable
   - `hubspotSyncedAt: text('hubspot_synced_at')` -- nullable

2. **src/db/index.ts** -- Add three PRAGMA statements on the `sqlite` instance BEFORE the `drizzle()` call. Insert these lines after `const sqlite = new Database(dbPath);`:
   ```
   sqlite.pragma('journal_mode = WAL');
   sqlite.pragma('busy_timeout = 5000');
   sqlite.pragma('synchronous = NORMAL');
   ```
   Do NOT export `sqlite` -- only `db` is exported (this is already the case, keep it).
