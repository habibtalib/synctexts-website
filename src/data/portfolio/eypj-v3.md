## The Challenge

Yayasan Pelajaran Johor administers loans, scholarships, grants, and training programmes for students across the state. The legacy ProLIMS system made it hard to manage the full application lifecycle, classify programmes consistently, and give applicants a modern self-service experience.

## Our Approach

We built eYPJ v3 as a multi-panel Laravel 12 application powered by FilamentPHP 3.3:

- **Admin panel** for complete system administration, application review, and analytics.
- **Applicant panel** with a friendly, multi-language (English / Bahasa Malaysia) application experience.
- **Program management** with categorized program types (loan, scholarship, training, grant), default requirements, fees, and criteria per type.
- **Smart classification** that maps legacy ProLIMS data into the new structure, plus integration with 27+ Malaysian banks for disbursement.
- **Application processing** with advanced filtering, quick approve/reject actions, bulk processing, and an analytics dashboard with trend charts.

## Results

- A single platform covering the entire application lifecycle from submission to disbursement.
- Clean separation between staff and applicant experiences, each themed and role-aware.
- Legacy data migrated and reclassified into a maintainable, modern schema.
