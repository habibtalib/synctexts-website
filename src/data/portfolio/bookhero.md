## The Challenge

BookHero needed an online storefront for custom book printing — letting customers configure and order print jobs while giving the team a modern back office to manage the catalogue and fulfilment.

## Our Approach

We built the platform on Laravel with an Inertia front end and a FilamentPHP 4 admin panel:

- **Customer ordering** for custom book-printing jobs.
- **Filament admin** for catalogue, orders, and fulfilment management, secured with role-based access (Filament Shield).
- **Modern UI** built with Inertia and Tailwind CSS 4.

## Results

- A self-service printing storefront for customers.
- A streamlined admin workflow for managing orders end to end.
- A clean, modern codebase on the latest Laravel/Filament stack.
